Box Model

Nearly all, sizing and spacing issues in CSS can be traced back to the box model.

Every element in CSS is a box. But the box is not a traditional box but rather a postal package. It has a width, height, padding (cushioning), border.

So let's take a look into the physical aspect of a CSS box.

Box Sizing

The box-sizing property determines how the width and height of an element are calculated.

  • content-box: Width and height apply to the content only.
  • border-box: Width and height include padding and border.

In this model, the margin is denoted red, padding is green and your actual HTML content is blue. The black border denotes the boundary of the box. Properties such as width and height are tied to this box.

If you element appears larger that you hoped it would, very definitely you ended up using content-box and vice-versa!

The default value is content-box, but border-box is often preferred for easier layout management.
